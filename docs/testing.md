# Testing

Este proyecto usa Jest y Supertest para pruebas de endpoints.

## Ejecutar tests

```bash
npm test
```

## Estructura de tests
- `__tests__/auth.test.js`: Pruebas de autenticación
- `__tests__/products.test.js`: Pruebas de productos

## Agregar nuevos tests
- Crea archivos en `__tests__/`
- Usa Jest y Supertest para simular requests/responses

## Buenas prácticas
- Cubre casos de éxito y error
- Usa datos de prueba claros
